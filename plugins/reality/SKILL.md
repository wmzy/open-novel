# 现实题材小说写作技能

你正在辅助创作一部中文现实题材长篇小说。本文件是写作质量的核心约束：它定义"什么是好"，并给出可操作的诊断方法。每一条技法都配有"差→好"对照范文——请照此校准你的输出。

现实题材的力量在于**精准**——精准地捕捉被时代裹挟的具体的人，精准地刺破日常的麻木。它不靠奇观，而靠**陌生化**与**白描**：让读者重新看见习以为常之物背后的荒诞、辛酸或温情。本文件先立通则，再谈现实题材独有之技。

---

## 一、核心写作原则（通则）

### 1. Show, Don't Tell
不报告情绪与判断，只用动作、物件、对话泄露。现实题材最忌评论员式的腔调——把判断权交给读者。

> **差**：老周下岗后非常沮丧，他觉得生活失去了意义，整个人都颓废了。
>
> **好**：老周把工牌塞进抽屉最里面，又拿出来，摩挲了一会照片。那是进厂第一年拍的，他穿着崭新的工作服，笑得牙都露着。他把工牌放回去，关上抽屉，坐到天黑没开灯。

### 2. 感官沉浸
现实藏在**琐碎的物质细节**里：菜价、水电、油烟、排队。这些细节本身就是阶层、处境与时代的刻度。

> **差**：菜市场的早市非常热闹，充满了浓厚的生活气息。
>
> **好**：王婶把手伸进那堆西红柿里，一个个捏过去。硬的要，软的不要。指甲掐下去能冒出水的才新鲜。她挑了二十分钟，省下三毛钱，回家路上一直攥着那几个西红柿，怕磕了。

### 3. 对话与节奏
现实对话是**口语化的、不完整的、有潜台词的**。人物不会说漂亮话，会说半句、会岔开、会沉默。短句白描，少用长定语。

> **差**："孩子，妈妈这是为了你好，你要理解妈妈的良苦用心啊！"她语重心长地说。
>
> **好**：
> "作业写完没？"
> "嗯。"
> "嗯是写完没写完？"
> 孩子没抬头。她把牛奶搁在桌角，凉了也没人喝。

## 二、反 AI 味诊断（现实题材版）

LLM 写现实题材最易犯的六种模式，成稿前逐条自检，命中即改。

### 模式 1：抽象情绪标签
> **差**：面对生活的重压，她感到无比的疲惫与迷茫，心中充满了无奈。
>
> **好**：她数了三遍工资条上的数字，把信封对折，塞进口袋。路过橱窗，玻璃上映出她的脸，她没敢多看。

### 模式 2：模板心理独白
> **差**：望着窗外的万家灯火，他不禁陷入了沉思，感慨着命运的不公与人生的无奈。
>
> **好**：楼下的灯一盏盏亮起来，又灭。他数到第十四盏的时候，烟烧到了手指。烫。他才想起自己根本没抽。

### 模式 3：排比堆砌
> **差**：这座城市既繁华又冷漠，既充满希望又令人绝望，既古老又现代。
>
> **好**：城是新的，路也是新的。新到连行道树都还没长出像样的荫凉。

### 模式 4：万能形容词
> **差**：这是一个典型的城乡结合部，充满了复杂的社会问题和鲜明的人间百态。
>
> **好**：出了地铁口，是两排还没拆的平房。墙上刷着"拆"字，红漆已经褪成粉色。一只野猫从垃圾堆里窜出来，叼着半条鱼。

### 模式 5：转折连词滥用
> **差**：尽管生活非常艰难，但是她依然没有放弃希望，然而现实的打击却接连不断。
>
> **好**：日子难。她没说过难。只是酱油用完了，兑了点水，又用了一周。

### 模式 6：情节概括代替演绎
> **差**：经过一番激烈的争吵，夫妻俩最终闹得不欢而散，矛盾进一步激化。
>
> **好**："钱呢？"她把碗往池子里一丢，水溅了一台面。
> 他没答，去够抹布。
> "我问你钱呢。"
> 抹布在台面上画了半个圈。他把水擦干净，把抹布叠好，搁回原处。一个字没说。她盯着他的背影，忽然就不想问了。

## 三、现实题材特有技法

### 1. 陌生化（Defamiliarization）
把读者习以为常的事物**写出陌生的质感**，迫使他们重新打量。陌生化不是故弄玄虚，而是用一个**非惯常的视角或细节**，让日常显出它的荒诞或锋利。

> **差**：上班族每天挤地铁上班，生活枯燥而乏味。
>
> **好**：地铁门一开，人像被吸进去。没有人说话，所有人都低头，脸被手机照得发蓝。她夹在两个人中间，脚几乎离了地，随着车厢的晃动，像一片挂在枝头的叶子，不知哪阵风能把她摘下来。

### 2. 社会细节即批判
不必直陈"社会不公"，让**一个具体的物件、一笔账、一道程序**替你说话。最锋利的批判是读者自己从细节里读出来的。

> **差**：医疗资源分配不均，底层百姓看病难、看病贵，这实在是一个严峻的社会问题。
>
> **好**：挂号窗口前排了三条队。她排错了一条，又重排。轮到她时，窗口里的声音隔着玻璃，闷闷的："专家号没了，普通号下午两点。"她看了眼手机，九点四十。她在医院门口的台阶上坐到下午，没吃饭，怕错过叫号。

### 3. 角色原型与处境
普通人、边缘人、理想主义者、现实主义者——原型是骨架，但**血肉必须是具体的处境**。一个角色之所以立得住，是因为他面对某件具体的事时，做出了只能由他做出的选择。

> 老李在厂里干了三十年。厂子要改制，让他签字买断工龄。三千块一年，三十年，九万。他拿着笔，手抖。窗外是他亲手栽的那排杨树，现在比厂房还高。他签了。回家路上，他把九万的存折贴身放着，一路上摸了七八回。

### 4. 结构：多线与时代感
现实题材常以**多线交织**呈现社会的全貌，让不同阶层、立场的人物命运相互映照。时代变迁可通过**同一空间的不同时间**来呈现（旧厂区改造、老街拆迁）。

## 四、阶段工作流与质量检查

六阶段（concept→world→characters→outline→scenes→writing），保存文件与推进 stage 由系统指令完成；**你的职责是在进入下一阶段前核对质量清单**。

- **concept**：核心冲突是否根植于**具体的现实处境**（生计、阶层、伦理、体制），而非抽象命题？两难抉择两头都有代价？
- **world**：时代背景有无**可触摸的物质细节**（物价、工种、方言、习俗）？社会规则（明规则与潜规则）是否清晰？
- **characters**：角色是否**典型又独特**（代表一群人，但只能是他自己）？动机是否符合其阶层与处境？
- **outline**：多线是否**交织呼应**而非平行无关？有无揭示时代变迁的节点？
- **scenes**：每场戏是否服务于**主题或人物处境**？"这场不写，读者会漏掉什么？"
- **writing**：通过"反 AI 味六模式"自检；每场景两种以上感官；对话口语化有潜台词；批判藏在细节里、不靠议论；角色状态与 `state.json` 一致；**每章后写 `第N章.summary.md` 并更新 `state.json`**。

## 五、文件结构说明

所有内容存放于项目目录的 `.novel/` 下，与通用小说相同：

| 文件 | 用途 |
|---|---|
| `concept.md` | 故事概念（现实冲突、社会主题、两难） |
| `world-building.md` | 时代背景、社会规则、物质细节、人物关系网 |
| `characters/profiles.md` | 角色档案（阶层、处境、动机、弧线） |
| `outline.md` | 大纲（三幕 + 分章） |
| `scenes.md` | 场景设计（冲突节拍、视点、目的） |
| `chapters/第N章.md` | 章节正文 |
| `chapters/第N章.summary.md` | **第 N 章压缩摘要**（≤200 字，每章必写） |
| `state.json` | 全局状态：角色位置/情绪/已知/关系、时间线、活跃伏笔、`lastUpdatedChapter` |
| `foreshadow.json` | 伏笔追踪 |
| `summary.md` | 全书摘要 |

现实题材可在 `world-building.md` 中专设"时代背景""社会规则""人物关系网"小节，无需另建目录。
