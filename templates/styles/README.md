# 文风参考目录

本目录存放 Markdown 格式的文风参考文件。AI 在写作章节正文时，会根据索引按需读取相关参考。

## 使用方式
1. 创建 Markdown 文件描述你希望的文风（如 tight-prose.md、lyrical.md、action-heavy.md）
2. 每个文件包含：风格特征、句式示例、节奏要求、用词偏好
3. 在 .novel/styles/index.md 中登记可用参考

## 文件格式建议
```markdown
# 风格名称

## 风格特征
（一句话概括）

## 句式要求
（具体要求）

## 节奏
（快/慢/交替）

## 用词偏好
（推荐/避免的词汇类型）

## 示例段落
（一段体现此风格的范文）
```
